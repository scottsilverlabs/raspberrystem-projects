Controlling Our Character

As mentioned earlier, the RaspberrySTEM has an enhanced Minecraft API (mcpi) that allows us to control our Minecraft character (Steve) and manipulate the Minecraft world. One of the things the API allows us to do is to remap the keyboard presses in other ways. For example, normally when we press the "W" key on the keyboard, Steve moves forward. But, using the API, we could change this -- we could make the "P" key on the keyboard move Steve forward instead. Or, better yet, we could control the forward movement of Steve using something else entirely...like a button on our RaspberrySTEM kit!

Below, we'll discuss the major actions our character can perform in the game, and how we can remap those actions to things other than the standard keyboard presses.
