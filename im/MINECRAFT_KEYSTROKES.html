The RaspberrySTEM has an enhanced Minecraft API (mcpi) that we'll discuss in an upcoming tutorial. In the meantime, let's jump in and use that API to see the types of things it allows us to do in the Minecraft world. One of the features of the API is that you can implement the functionality to move your character (Steve) around, exactly as if you would have pressed the W, S, A or D keys on the keyboard (these are the keys that move Steve Forward, Backward, Left & Right).

Once control is imported, you can control Steve with the functions forward(), backward(), left() and right(). Calling those functions without arguments is just like pressing and holding the corresponding key (W, S, A, or D) on the keyboard. The key is not released until you call the function with the argument release=True.
