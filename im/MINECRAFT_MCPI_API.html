The Software

MCPI stands for Minecraft Pi. The mcpi API is a set of functions to query and control the player in Minecraft. Note that there is an mcpi API downloadable online, which is slightly different from the one available on the RaspberrySTEM. The RaspberrySTEM has an enhanced version that includes functions for controlling the movement of the Minecraft player, determining the direction the player is looking, and showing/hiding the Minecraft window.
