As we mentioned (and saw) in the "Hello, Minecraft!" project, Minecraft has its own API that can be used to control and manipulate the Minecraft world. The API is called mcpi, which stands for Minecraft Pi. The mcpi API is a set of functions to query and control the player in Minecraft.

Note: There is an mcpi API downloadable online, which is slightly different from the one available on the RaspberrySTEM. The RaspberrySTEM has an enhanced version that includes functions for controlling the movement of the Minecraft player, determining the direction the player is looking, and showing/hiding the Minecraft window. When using the mcpi API on the RaspberrySTEM, we highly recommend using our API, as opposed to the standard mcpi API.
